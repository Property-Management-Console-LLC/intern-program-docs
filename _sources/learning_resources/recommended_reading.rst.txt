Recommended Reading
===================

Books, articles, and documentation that will deepen your understanding of our technologies.

1. **Java & Spring Boot**  
   - **Book: Spring in Action by Craig Walls**  
     A highly recommended book to get a thorough understanding of the Spring Framework.  
     **Link**: [https://www.manning.com/books/spring-in-action-fifth-edition](https://www.manning.com/books/spring-in-action-fifth-edition)
   
   - **Documentation: Official Spring Boot Documentation**  
     Always refer to the official Spring documentation for the most accurate and up-to-date information.  
     **Link**: [https://docs.spring.io/spring-boot/docs/current/reference/htmlsingle/](https://docs.spring.io/spring-boot/docs/current/reference/htmlsingle/)

   - **Article: REST API Design Best Practices**  
     A great article that covers REST API design principles.  
     **Link**: [https://www.freecodecamp.org/news/rest-api-design-best-practices-build-a-rest-api/](https://www.freecodecamp.org/news/rest-api-design-best-practices-build-a-rest-api/)

2. **Angular**  
   - **Book: Angular Up & Running by Shyam Seshadri**  
     A solid resource for learning Angular, covering the basics to advanced features.  
     **Link**: [https://www.oreilly.com/library/view/angular-up-and/9781491999837/](https://www.oreilly.com/library/view/angular-up-and/9781491999837/)
   
   - **Documentation: Angular Official Documentation**  
     The official guide to mastering Angular and its core concepts.  
     **Link**: [https://angular.io/docs](https://angular.io/docs)

3. **Flutter**  
   - **Book: Flutter for Beginners by Alessandro Biessek**  
     A beginner-friendly book that covers the basics of Flutter and Dart.  
     **Link**: [https://www.packtpub.com/product/flutter-for-beginners/9781838647773](https://www.packtpub.com/product/flutter-for-beginners/9781838647773)
   
   - **Documentation: Flutter Official Documentation**  
     Learn about the Flutter SDK and tools from the official docs.  
     **Link**: [https://flutter.dev/docs](https://flutter.dev/docs)

4. **Python (AI Focus)**  
   - **Book: Python Machine Learning by Sebastian Raschka**  
     One of the most popular books to get started with machine learning using Python.  
     **Link**: [https://www.packtpub.com/product/python-machine-learning-third-edition/9781789955750](https://www.packtpub.com/product/python-machine-learning-third-edition/9781789955750)
