Workstation Setup
=================

This section provides guidance on setting up your workstation with the necessary software and resources for your team. Follow the steps below to ensure your system is ready for productive work.

1. **General Setup for All Interns**
   ---------------------------------
   Every intern should install the following tools and resources:
   
   - **Git**: Used for version control.
     - Download: [Git Official Site](https://git-scm.com/)
     - Installation Guide: After downloading, follow the setup instructions to complete the installation.
   
   - **Visual Studio Code**: A free and powerful text editor for coding in multiple languages.
     - Download: [Visual Studio Code](https://code.visualstudio.com/)

   - **Google Chrome**: Preferred browser for accessing our online resources and documentation.
     - Download: [Google Chrome](https://www.google.com/chrome/)

   - **Zoom**: For team meetings and company updates.
     - Download: [Zoom](https://zoom.us/download)

2. **Frontend and User Experience (FUX) Team**
   -------------------------------------------
   As a member of the Frontend team, youâ€™ll primarily work with Angular and other web development tools.

   - **Node.js and npm**: Required to run Angular applications.
     - Download: [Node.js](https://nodejs.org/)
     - Check if npm installs automatically with Node.js. If not, refer to npm installation guide on the same website.
   
   - **Angular CLI**: Command line tool for Angular.
     - Installation: After installing Node.js, open a terminal and run:
       ```
       npm install -g @angular/cli
       ```

   - **Figma**: For UI/UX design and prototyping.
     - Download: [Figma](https://www.figma.com/downloads/) (Free version available for individual accounts)

3. **Backend and API Development (BAD) Team**
   ------------------------------------------
   The Backend team primarily uses Java and Spring Boot for API development.

   - **Java Development Kit (JDK)**: Install JDK 17 or higher for compatibility with Spring Boot.
     - Download: [OpenJDK 17](https://jdk.java.net/17/)
   
   - **IntelliJ IDEA (Community Edition)**: Free IDE for Java development.
     - Download: [IntelliJ IDEA](https://www.jetbrains.com/idea/download/) (Choose the Community version)
   
   - **Postman**: For testing API endpoints.
     - Download: [Postman](https://www.postman.com/downloads/)

4. **DevOps, Infrastructure, and Data (DID) Team**
   -----------------------------------------------
   The DevOps team is responsible for managing infrastructure, pipelines, and data resources.

   - **Docker Desktop**: For containerizing applications.
     - Download: [Docker Desktop](https://www.docker.com/products/docker-desktop/)
     - Note: Docker Desktop is free for personal use but may require an account.

   - **AWS CLI**: Interface for managing AWS services.
     - Download: [AWS CLI](https://aws.amazon.com/cli/)

   - **Terraform**: Used for infrastructure as code.
     - Download: [Terraform](https://www.terraform.io/downloads.html)
   
   - **DBeaver**: Free database management tool for managing data resources.
     - Download: [DBeaver Community Edition](https://dbeaver.io/download/)

5. **Full Stack Development (FSD) Team**
   -------------------------------------
   Full Stack developers work across both frontend and backend technologies, so you'll need resources for both environments.

   - **Node.js, npm, and Angular CLI**: Follow the instructions under "Frontend and User Experience Team."
   
   - **Java and IntelliJ IDEA**: Follow the instructions under "Backend and API Development Team."
   
   - **MySql**: GUI for database management (useful if your projects involve relational databases).
     - Download: [MySql Workbench](https://dev.mysql.com/downloads/workbench/)

6. **Python and AI Module Development**
   ------------------------------------
   Python is used for AI and machine learning components. 

   - **Python 3.x**: Install the latest version.
     - Download: [Python](https://www.python.org/downloads/)

   - **PyCharm (Community Edition)**: Free IDE for Python development.
     - Download: [PyCharm](https://www.jetbrains.com/pycharm/download/)

   - **Jupyter Notebook**: Great for data science and AI projects.
     - Installation: Once Python is installed, open a terminal and run:
       ```
       pip install jupyter
       ```

**Additional Notes**
--------------------
- Ensure that you update your PATH environment variable if prompted during installation, particularly for command-line tools like Node.js, Java, and Python.
- Always check for any additional team-specific instructions from your mentor or lead, as they may have specific preferences for certain versions or configurations.
