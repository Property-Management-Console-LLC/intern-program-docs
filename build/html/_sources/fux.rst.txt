# Frontend and User Experience (FUX)
Welcome to the FUX team! Our mission is to design intuitive and user-friendly interfaces that deliver a seamless experience to our users.


## Development Stack
Our stack is designed for performance and user-centric design:
- **Framework**: Angular
- **Languages**: JavaScript, TypeScript, Flutter
- **Styling**: CSS, SCSS, SASS
- **UI Libraries**: Bootstrap, Material UI
- **Design Tools**: Figma


## Onboarding Process
To get started:
1. Clone the repository from GitHub.
2. Install Node.js and Angular CLI.


## Team Standards & Guidelines
- Follow the company's UI/UX guidelines.
- Keep components small and reusable.
- Ensure code follows our linter rules (Prettier, ESLint).
