Development Environments
========================

Our projects span across different platforms, each requiring a specific setup. Below are instructions on setting up the development environments for the various technologies used in our company.

1. **Java & Spring Boot Development Environment:**
   - **IDE**: We recommend IntelliJ IDEA or Eclipse for working with Java.
   - **JDK**: Install Java Development Kit (JDK) version 11 or higher.
   - **Spring Boot**: Familiarize yourself with Spring Boot by following the official documentation: https://spring.io/guides/gs/spring-boot/.

   **Required Tools:**
   - Maven or Gradle for dependency management.
   - Postman or curl for testing APIs.

2. **Angular Development Environment:**
   - **IDE**: We recommend using Visual Studio Code for front-end development.
   - **Node.js**: Install Node.js and npm (Node Package Manager).
   - **Angular CLI**: Install Angular CLI by running:  
     ```
     npm install -g @angular/cli
     ```

   **Required Tools:**
   - Browser Developer Tools (Chrome or Firefox) for debugging.
   - A REST API client like Postman to test API integration.

3. **Flutter Development Environment:**
   - **IDE**: Use Android Studio or Visual Studio Code with Flutter plugins.
   - **Flutter SDK**: Install the Flutter SDK from https://flutter.dev/docs/get-started/install.
   - **Emulator/Physical Device**: Set up an Android emulator or use a physical device for testing.

   **Required Tools:**
   - Flutter DevTools for performance profiling and debugging.
   - Android SDK for Android development.

4. **Python Development Environment (for AI projects):**
   - **IDE**: Use PyCharm or Visual Studio Code with Python extensions.
   - **Python Version**: Install Python 3.8 or higher.
   - **Virtual Environment**: Set up a virtual environment using `venv` or `virtualenv`.

   **Required Tools:**
   - Jupyter notebooks for experimentation with AI models.
   - Popular libraries such as TensorFlow, PyTorch, or scikit-learn, depending on the project requirements.
